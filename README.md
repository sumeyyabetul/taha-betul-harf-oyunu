# 💕 Taha & Betül Harf Oyunu

Taha ve Betül için özel olarak hazırlanmış, iki kişilik romantik bir harf oyunu. Türk alfabesindeki harfler sırayla dağıtılır; oyuncular zamanında harf alarak puan kazanır. Bonus harfler, combo sistemi ve sevgi dolu mesajlarla eğlenceli bir deneyim sunar.

## 🎮 Oyun Nasıl Oynanır?

1. **Oyuna Başla** butonuna tıklayın.
2. Sırayla Taha ve Betül için harfler verilir.
3. Sırası gelen oyuncu **10 saniye** içinde **Harf Al** butonuna basmalıdır.
4. Harf alındıktan sonra **3 saniye** içinde sıra diğer oyuncuya geçer.
5. Tüm harfler bitince skorlara göre kazanan belirlenir.

## 🎯 Kurallar

- Her harf alındığında **1 puan** kazanılır.
- **⭐ Bonus harfler** 2 kat puan verir.
- **🔥 Combo:** Art arda kazanan oyuncu combo yapar ve ekstra puan alır.
- Süre dolmadan harf alınmazsa sıra karşı oyuncuya geçer.

## ✨ Özellikler

- Türk alfabesine uygun 29 harf
- Canlı skor takibi ve combo göstergesi
- Geri sayım ve ilerleme çubukları
- Rastgele sevgi sözleri
- Yüzen kalp animasyonları
- Arka plan müziği
- Mobil uyumlu tasarım

## 🚀 Nasıl Çalıştırılır?

Projeyi bilgisayarınıza indirdikten sonra `index.html` dosyasını çift tıklayarak tarayıcıda açmanız yeterlidir. Ek kurulum gerekmez.

```bash
# Repoyu klonlamak için
git clone https://github.com/sumeyyabetul/taha-betul-harf-oyunu.git
cd taha-betul-harf-oyunu
```

Ardından `index.html` dosyasını tarayıcıda açın.
